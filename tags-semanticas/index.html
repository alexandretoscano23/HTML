O HTML

HTML é a sigla para HyperText Markup Language, ou Linguagem de Marcação de Hipertexto em português. É a linguagem padrão utilizada para criar a estrutura e o conteúdo de páginas da web.


Pode Conter:

  • Tags para motores de busca
  • Tags para adicionar elementos visíveis no seu site
  • Links para diversos sites ou arquivos

Em resumo, o HTML é a linguagem fundamental para contruir páginas web. Ele fornece as ferramentas básicas para criar a estrutura e o conteúdo de qualquer site que você visita na internet.
